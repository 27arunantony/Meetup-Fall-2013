Writing to a file

In [11]: 
text = "When you are courting a nice girl, an hour seems like a second.\n"
text += "When you sit on a red-hot cinder, a second seems like an hour.\n"
text += "That's relativity."

Create the outfile object.

In [12]: 
outfile = open('albert.txt','w')

Write the text data.

In [13]: 
outfile.write(text)

Close the outfile object.

In [14]: 
outfile.close()

Reading files

In [16]: 
infile = open('albert.txt','r')
text = infile.read()
infile.close()

print text

When you are courting a nice girl, an hour seems like a second.
When you sit on a red-hot cinder, a second seems like an hour.
That's relativity.

The with Statement

Peforms a close() for you.

In [17]: 
with open('albert.txt','r') as infile:
    text = infile.read()
In [18]:

In [19]: 
with open('albert.txt','w') as outfile:
    outfile.write(text[:21])
    outfile.write('\n')

JSON

More flexible than CSV for structured text

In [47]: 
data = {'name': 'Guido van Rossum',
        'language': 'Python',
        'similar': ['c','Modula-3','lisp'],
        'users': 1000000}
In [49]: 
import json
json.dumps(data)
Out[49]:
'{"similar": ["c", "Modula-3", "lisp"], "name": "Guido van Rossum", "language": "Python", "users": 1000000}'
In [63]: 
with open('data.json','w') as outfile:
    outfile.write(json.dumps(data))

Read JSON

In [124]: 
with open('data.json','r') as infile:
    text = infile.read()
    data = json.loads(text)
In [125]: 
print data

{u'similar': [u'c', u'Modula-3', u'lisp'], u'name': u'Guido van Rossum', u'language': u'Python', u'users': 1000000}
In [126]: 
print data['name']

Guido van Rossum

Take a minute to think about how powerful and simple this is.

Pickle serialization

Stores data efficiently in binary format.

In [127]: 
import pickle
In [131]: 
with open('data.pickle', 'wb') as output:
    pickle.dump(data, output)

Read it back in.

In [132]: 
with open('data.pickle', 'rb') as input:
    data = pickle.load(input)
In [133]: 
print data['name']

Guido van Rossum
