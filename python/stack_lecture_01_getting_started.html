What is Python?

Python is a modern, general-purpose, object-oriented, programming language.

  • Simple, clean syntax
  • Easy to learn
  • Interpreted
  • Dynamically typed
  • Runs everywhere: Linux, Mac, and Windows
  • Free, as in beer and speach
  • Expressive: do more with fewer lines of code
  • Lean: modules

History

There is only one way to do it

  • Created by Dutch programmer Guido van Rossum, 1991
  • Provided administrative scripting for the Amoeba distributed OS.
  • Declared Dictator for Life by legions of followers.
  • Syntax from C
  • lambda, reduce, filter, and map
    • a Lisp hacker who missed them
  • Modula-3 inspired keyword arguments and imports

Performance

The greatest performance improvement comes from going to the working state from the non-working state

  • Minimal time required to develop code
  • Rapid protoytping
  • Interpreted and dynamically typed means it's slower
    • Use optimized libraries: e.g. numpy , scipy
    • Find the bottleneck and write it in C/C++/Fortran: e.g. f2py, cython
    • Just-in-time: e.g. numba
  • Implementing the built-in Python modules would require some advanced programming skills in other languages

Advantages

  • Interpretive
  • Named function arguments
  • Heterogeneous data structures
    • dictionary
    • lists
  • No memory management
  • Print structures
  • Packages: large community of support
  • Modular: great for larger projects (modules)

Modules for Data Analysis

Base
  • ipython: interactive shell
  • numpy : array and matrix library
  • scipy : scientific libraries
  • matplotlib : visualization
  • pandas : data frames
Additional
  • Requests: data from the web
  • Beautiful Soup: parse web pages
  • pytables : fast, binary files
  • sqlite3: binding for sqlite3
  • spark: in-memory MapReduce
  • disco: MapReduce

Modules for High Performance Computing

  • ipython parallel: Interactive MPI, direct, and load balanced
  • mpi4py : MPI Bindings
  • cython.parallel : OpenMP
  • multiprocessing: Quick parallel tasks
  • pycuda: GPU
  • Accelerate: GPU
